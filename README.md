## Getting Started

The application is divided into two main sections: **Servers** and **Worlds**. Both require the world or server to have its respective **Addon** installed.

![Tabs Overview](assets/image.png)

---

## üåç Mode: Worlds (P2P)

Ideal for playing with friends in local worlds via direct connection.

### For the Host:
1. Go to the **Worlds** tab and select the **My World** option.
   ![My World Section](assets/myworld.jpeg)
2. Once inside the room, you will see a **Link** at the top.
   ![Command Link](assets/link.jpeg)
3. Click the link to automatically copy the command to your clipboard.
4. **In Minecraft:** Paste the command into your world's chat.
   > ‚ö†Ô∏è **Critical Requirements:**
   > * You must have **cheats enabled**.
   > * In your Minecraft **Profile** settings, the **"Enable WebSocket"** option must be toggled on.

### For Guests:
1. In the **Worlds** tab, tap the **Add Friend** (+) button located at the bottom right.
   ![Add Button](assets/add.jpeg)
2. Add the host by their gamertag. Once added, click on their name in the list.
3. You will enter a common room where you can see all participants. 
4. **You're all set!** No further action is needed in the app; just join your friend's world in Minecraft.

---

## üñ•Ô∏è Mode: Servers

Configuration for communities and dedicated servers.

1. Go to the **Servers** tab and press the add button.
2. Enter the **IP** and **Port** of the server you wish to connect to.
3. Upon entering the server room, all members will see a **Link** at the top.
4. **Activation:** Only **one user** needs to copy the link and paste the command into the Minecraft server chat.
   * By doing this, all other users in the room will connect automatically.
   * *Note:* If all players leave the server, the first one to re-enter must run the command again.
   * **Key Difference:** In this mode, it is **not necessary** to enable WebSockets in your Minecraft profile.
